# End-to-End Retail Data Modeling & Analytics Pipeline

## Overview
This project implements an **end-to-end retail data modeling and analytics pipeline** using **Databricks**, **PySpark**, **SQL**, and **Delta Lake**, following the **Medallion Architecture (Bronze, Silver, Gold)**.  
The pipeline processes raw retail transactional data and transforms it into **analytics-ready datasets** for reporting, KPI tracking, and business insights.

---

## Architecture
The solution follows the **Lakehouse Medallion Architecture**, ensuring scalability, reliability, and maintainability.
